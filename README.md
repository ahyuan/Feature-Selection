# Feature Selection

This repository collects feature selection implementations and related paper resources.

Project URL: <https://github.com/ahyuan/Feature-Selection>

Related CNAFS project URL: <https://github.com/ahyuan/CNAFS>

## Methods

- [CNAFS](./CNAFS/) - Convex Non-Negative Matrix Factorization With Adaptive Graph for Unsupervised Feature Selection.
- [MGAGR](./MGAGR/) - Robust Unsupervised Feature Selection via Multi-Group Adaptive Graph Representation.

## Repository Contents

- `CNAFS/` - MATLAB implementation, usage notes, paper information, experimental summary, and BibTeX for CNAFS.
- `MGAGR/` - MATLAB implementation, usage notes, paper information, experimental summary, result-table screenshot, and BibTeX for MGAGR.
- `LICENSE` - repository license.

The repository contains core algorithm implementations only. Benchmark datasets, preprocessing scripts, parameter-search scripts, and evaluation scripts are not included.
