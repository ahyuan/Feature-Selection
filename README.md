# Feature Selection

This repository collects feature selection implementations and related paper resources.

Project URL: <https://github.com/ahyuan/Feature-Selection>

Related CNAFS project URL: <https://github.com/ahyuan/CNAFS>

## Methods

- [CNAFS](./CNAFS/) - Convex Non-Negative Matrix Factorization With Adaptive Graph for Unsupervised Feature Selection.
- [MGAGR](./MGAGR/) - Robust Unsupervised Feature Selection via Multi-Group Adaptive Graph Representation.
- [SSDS](./SSDS/) - Symmetrical Self-Representation and Data-Grouping Strategy for Unsupervised Feature Selection.

## Repository Contents

- `CNAFS/` - MATLAB implementation, usage notes, paper information, experimental summary, and BibTeX for CNAFS.
- `MGAGR/` - MATLAB implementation, usage notes, paper information, experimental summary, cropped result-table screenshot, and BibTeX for MGAGR.
- `SSDS/` - MATLAB implementation, reproduction live script, experimental result files, cropped result-table screenshots, and BibTeX for SSDS.
- `LICENSE` - repository license.

Benchmark datasets are not included. See each method directory for the available MATLAB implementation files, experiment notes, and paper resources.
